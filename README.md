# 🚀 AI Resume Enhancer

AI Resume Enhancer is a Streamlit web application powered by Google Gemini AI that analyzes resumes against job descriptions and provides instant feedback to improve job application success.

---

## 📌 Features

- 📄 Upload Resume (PDF/Text)
- 💼 Paste Job Description
- 📊 ATS Match Score
- 💡 Strengths Analysis
- ⚠ Missing Skills Detection
- 🚀 Improvement Suggestions
- 📄 AI-Generated Cover Letter

---

## 🧠 How It Works

The app uses Google Gemini AI to compare your resume with a job description and returns structured insights including skill gaps, strengths, and ATS-style scoring to help improve your resume.

---

## ⚙️ Tech Stack

- Python
- Streamlit
- Google Generative AI (Gemini)
- PyPDF2
- python-dotenv
